This is a sample blog post to show you the structure.

## Writing Posts

1. Create a new `.md` file in the `content/` folder
2. Add frontmatter with `title`, `created`, and `tags`
3. Write your content in Markdown
4. Push to GitHub and it will auto-deploy

## Features

- Minimalist design
- Light/dark mode (auto-switches based on system preference)
- RSS feed at `/index.xml`
- Tags and categories
- Chronological listing
- Plausible analytics

Enjoy writing!
